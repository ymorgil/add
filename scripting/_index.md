# 🖥️ Repositorio de Scripting

## Descripción

Este repositorio es un **repositorio base de scripting**, pensado como punto de partida para el alumnado de
**Administración de Sistemas Informáticos en Red (ASIR -- Grado Superior)** y para cualquier persona interesada en aprender scripting aplicado a la administración de sistemas.

La idea de este repositorio es servir como:

-   📘 Material de apoyo para clase
-   🧪 Zona de pruebas y experimentación
-   🛠️ Colección de ejemplos reales
-   ✍️ Espacio colaborativo con aportaciones del alumnado

Aquí se recopilan **scripts, ejercicios, ejemplos y propuestas**, desde un nivel básico hasta un nivel más avanzado,  siempre con un enfoque **práctico y realista**.

## Estructura del repositorio

### Bash 🐧

Scripts orientados a entornos **GNU/Linux** y servidores.

Contenidos habituales: - Automatización de tareas - Gestión de usuarios y permisos - Procesos y servicios - Copias de seguridad - Logs y mantenimiento del sistema

📁 Carpeta: `Bash/`



### PowerShell 🪟

Scripting enfocado a **Windows y entornos híbridos**.

Contenidos habituales: - Administración del sistema - Active Directory - Servicios y procesos - Automatización administrativa - Gestión remota

📁 Carpeta: `PowerShell/`



### Python 🐍

Lenguaje de scripting **multiplataforma**, muy usado en sistemas, DevOps y automatización.

Contenidos habituales: - Automatización de tareas - Gestión de archivos y directorios - Scripts de administración - Introducción a scripting avanzado - Buenas prácticas y estructura de código

📁 Carpeta: `Python/`



## Enfoque educativo

Este repositorio está diseñado con un enfoque **didáctico**,
priorizando:

-   Claridad del código
-   Comentarios explicativos
-   Ejercicios progresivos
-   Aplicación práctica en sistemas reales

Está pensado para **aprender haciendo**, no solo para leer código.


## Contribuciones

Se aceptan y se fomentan las contribuciones, especialmente del alumnado.

Puedes contribuir con: - Nuevos scripts - Ejercicios - Soluciones - Mejoras de código - Documentación

La idea es que el repositorio **crezca con el uso en el aula**.
